HTML & CSS

Seit dem Erscheinen von HTML und CSS haben diese Technologien eine lange Entwicklung vollzogen. Was anfangs zur statischen Anzeige von Dokumenten gedacht war, hat seit dem Erscheinen von HTML5 und CSS3 seinen Siegeszug durch das Web angetreten. Mittlerweile werden diese Technologien nicht nur im klassischen Web eingesetzt, sondern finden auch Einzug in Bereichen wie Healthcare und Automotive. Web-Apps stehen heutzutage nicht mehr im Schatten nativer Apps, auch dank HTML5 und CSS3. Umso wichtiger ist es deshalb diese grundlegenden Technologien zu verstehen und zu beherrschen.
